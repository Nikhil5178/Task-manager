# Task Manager App

Full-stack task management app with JWT auth and MongoDB.

## Setup

### Backend
```bash
cd server
cp .env.example .env
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

### Environment Variables (.env)
- MONGO_URI
- JWT_SECRET

## Features
- User registration/login
- Add/view tasks (protected)
- MongoDB + Express + React
